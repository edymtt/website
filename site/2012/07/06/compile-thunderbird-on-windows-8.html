Please note that these instructions are obsolete and I keep them for historical reasons, for compilation on Windows 8 you should refer to these instructions on Mozilla Developer Network

  1. download Visual C++ Express 2010
  2. apply Visual Studio 2010 SP1
  3. install Windows 8 SDK (this includes SDK 7.0A)
  4. install DirectX SDK (this will install .NET 2.0/3.5 alongside)
  5. install MozillaBuild
  6. start MozillaBuild from start-msvc10.bat
  7. follow the instructions to obtain the Daily executable
